About ESP32JTAG

ESP32JTAG is an open-source, Wi-Fi-enabled JTAG debugger that consolidates multiple tools—JTAG/SWD debugging, a 16-channel logic analyzer, FPGA configuration, and UART terminal access—into one compact, wireless device. It frees you from the clutter and limitations of traditional wired probes by providing a built-in web server for driver-free operation.

It's designed to streamline the workflow for embedded engineers, allowing them to debug and develop from any device with a web browser, anywhere on their Wi-Fi network.

Getting Started

  1. Power On: Connect ESP32JTAG to your computer or a 5V power source using the USB Type-C port.
  2. Connect to Wi-Fi: The device will create a temporary Wi-Fi access point (AP) named ESP32JTAG-XXXX. Connect to it using your computer or phone.
  3. Configure Wi-Fi: Open a browser and navigate to http://192.168.4.1 to configure the device to connect to your local network. You can also use the LCD screen to view its current IP address.
  4. Start Debugging: Once connected to your network, the LCD will display the new IP address. Use this address to access the web server and configure your debugging session with tools like OpenOCD, GDBServer, or Blackmagic Probe.

Key Features

  • All-in-One Tool: Combines JTAG/SWD debugging, 16-channel logic analyzer, FPGA programmer, and UART terminal.
  • Wireless & Driver-Free: Connect via Wi-Fi 6 or Bluetooth 5.0. No drivers needed thanks to the built-in web server.
  • Versatile Connectivity: Supports popular open-source tools like OpenOCD, Blackmagic Probe, and openFPGALoader.
  • Open Hardware & Software: Schematics, BOM, and firmware are open-source, allowing for community collaboration and customization.

Troubleshooting

  • Can't connect to the Wi-Fi AP? Ensure the device is powered on. If the LCD is off, check the USB cable. The AP is only active during initial setup or if it fails to connect to a configured network.
  • Can't access the web server? Make sure you are on the same Wi-Fi network as the ESP32JTAG. Check the IP address on the LCD screen. Try clearing your browser's cache or using an incognito window.
  • The device is not being detected by my IDE? Ensure you've configured your IDE (e.g., VSCode) to connect to the ESP32JTAG's IP address and the correct port for the debugging protocol (e.g., OpenOCD).

Technical Specifications

Feature Details
Processor Dual-core ESP32-S3 @ 266 MHz
Memory 16 MB flash, 8 MB PSRAM
Connectivity Wi-Fi 6, Bluetooth 5.0, USB Type-C
I/O 4 configurable 4-wire ports
Supported Modes MCU JTAG/SWD, FPGA JTAG, UART, Logic Analyzer
Dimensions 33 × 40 × 5 mm
